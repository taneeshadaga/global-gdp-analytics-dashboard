# Global GDP Analytics Dashboard

An interactive Streamlit dashboard for exploring global GDP growth trends, comparing countries and regions, and experimenting with simple forecasting and policy simulation tools.

## Overview

This project analyzes GDP growth data across countries over time. The goal is to make macroeconomic trends easier to explore through interactive visualizations, summary statistics, forecasting models, and basic “what-if” policy simulations.

This project was built as both a data analytics portfolio project and a way to practice using AI-assisted development tools like Windsurf to improve development speed, code organization, and project iteration.

## Features

- Interactive GDP growth visualizations by country and region
- Summary statistics for selected time periods
- Top and bottom GDP growth performers
- Regional comparison charts
- GDP growth volatility analysis
- Forecasting tools using ARIMA, Prophet, and XGBoost
- Simple tariff policy impact simulation
- Automated insights and downloadable reports

Software Used: Python, Streamlit
Libraries: pandas, numpy, plotly, scikit-learn, statsmodels, Prophet, XGBoost, ReportLab

**Note:** Forecasting and policy simulation features are simplified and intended for exploratory analysis.
